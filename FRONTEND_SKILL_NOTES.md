# Growly Frontend Skill Notes

Direction: serious premium growth brand, not generic SMM panel.

Principles:
1. Trust is repeated on every step, not only footer.
2. Preview before payment: account preview for followers, post preview for likes/views.
3. Use modern CSS progressively: cascade layers, scroll reveals, View Transition API fallback-safe.
4. Keep motion subtle: product-grade, not casino.
5. Footer is intentionally dense: policies, badges, proof, disclaimers.
6. Never ship real secrets in .env. Use .env.example only.

Important files:
- index.html: new 2026-level landing art direction
- search.html: account/package flow
- checkout.html: order summary
- api/profile.js: Instagram + TikTok profile lookup
- api/posts.js: lazy post lookup for optional recent posts
