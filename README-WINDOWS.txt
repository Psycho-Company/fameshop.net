WINDOWS ICIN EN KOLAY CALISTIRMA

1. growly-real-live-windows.zip dosyasini sag tikla
2. Tumunu ayikla / Extract All de
3. Cikan klasoru ac
4. start-windows.bat dosyasina cift tikla

Bu kadar.

- Direkt CMD acip npm install yazma. Yanlis klasorde acilir.
- npm install sadece package.json olan klasorde calisir.
- start-windows.bat otomatik dogru klasorde calistirir.
